Curso: Carreira Front End
Empresa: DevMedia
Desafio do Projeto: Usar a TAG Meta que foram aprendidas durante a aula e aplicação do Style dentro do HTML para estilizar a página.

meta charset="utf-8" - Para determinar qual conjunto de caracteres utlizado na Página. 

meta name="description" - Para colocar a descrição da página para os buscadores.

meta name="roboots com noindex, nofollow" - Para simular uma página interna que não deve ser exibida nos buscadores e ser associada a links externos.

meta name="viewport" content="width=device-width, initial-scale=1.0" - Para que a página adpte o conteúdo ao tamanho de tela usada pelo o usuário, definindo o zoom para o padrão do navegador.

title - Para definir o título da página no Navegador.

**As cores realmente foram bem aleatórias, apenas para testes.**
